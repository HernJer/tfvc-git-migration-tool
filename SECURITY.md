# Security Policy

## Supported Versions

Currently, only the latest release of `tfvc2git` receives security updates. 

| Version | Supported          |
| ------- | ------------------ |
| > 1.0   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `tfvc2git`, please do not open a public issue. Instead, please report it via GitHub Security Advisories or contact the repository owner directly.

We will endeavor to respond to your report within 48 hours and work with you to verify and resolve the issue.
