# Tfvc2Git Documentation

Welcome to the detailed documentation for the `tfvc2git` migration tool. This documentation is intended for administrators and developers who are performing complex migrations from TFVC (Team Foundation Version Control) to Git (e.g., GitHub, Azure Repos).

## Table of Contents

1. [Architecture & Pipeline](./architecture.md)
   * Deep dive into the 4-step migration pipeline (Export, Replay, Verify, Report).
2. [Configuration Guide](./configuration.md)
   * Detailed reference for `config.json`, including source mappings, branch dependencies, and performance tuning.
3. [Troubleshooting & Edge Cases](./troubleshooting-and-edge-cases.md)
   * A comprehensive guide to handling pipeline failures, network interruptions, remote push rejections, and TFVC edge cases.
